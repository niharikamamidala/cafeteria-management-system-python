# Cafeteria Management System (Python)

A command-line based Cafeteria Order Management System built using Python.

## Features
- Place normal and priority orders
- Order processing using Queue
- Priority order handling using Deque
- Completed order tracking
- Menu driven interface

## Data Structures Used
- Queue (FIFO order processing)
- Deque (priority orders)
- List (completed orders)
- Dictionary (order information)

## How to Run

1. Clone the repository
2. Run the program

python cafeteria_system.py

## Example Operations

1. Place Order
2. Process Order
3. View Completed Orders
4. Exit
